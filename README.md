# minime lovers

Tienda e-commerce de cosméticos colombianos. Venta de labiales en polvo, bolsos mini y kits personalizados. Desplegada en [minime.com.co](https://minime.com.co).

## Stack

- HTML/CSS/JavaScript vanilla (sin frameworks)
- Netlify (hosting + serverless functions)
- [Bold](https://commerce.bold.co) — pasarela de pago colombiana
- EmailJS — confirmaciones de pedido por correo
- Google Sheets — registro de pedidos vía Apps Script
